# 🚀 TaskFlow - Full Stack To-Do App

A modern productivity app built with React and Supabase.

## ✨ Features
- Add / Delete / Complete tasks
- Categories with autocomplete
- Priority levels (High, Medium, Low)
- Due dates + overdue highlighting
- Sorting (date & priority)
- Clean modern UI

## 🛠 Tech Stack
- React
- Supabase (Backend)
- CSS
